# Data-Driven Analysis of Bengaluru Civic Complaints for Urban Service Insights

## Project Overview

This project analyzes Bengaluru (BLR) municipality complaint data from the I Change My City platform to identify patterns in civic issues, complaint frequency, departmental workload, and resolution efficiency.

The analysis uses Python-based exploratory data analysis (EDA) techniques to uncover meaningful insights that can help understand urban infrastructure challenges and public service performance.

---

## Objectives

• Identify most common civic issues reported by citizens  
• Analyze complaint distribution across wards  
• Evaluate complaint resolution patterns  
• Identify departments handling most complaints  
• Discover trends in complaints over time  
• Detect seasonal patterns in civic issues  

---

## Dataset

Source:
I Change My City Open Data Portal

The dataset contains citizen-reported civic complaints including:

- Complaint category and subcategory
- Complaint status
- Department responsible
- Ward and location details
- Complaint timestamp
- Public engagement (comment count)
